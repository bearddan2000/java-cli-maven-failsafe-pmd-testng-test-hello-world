# java-cli-maven-failsafe-pmd-testng-test-hello-world

## Description
A POC for maven app using testng
framework with failsafe and pmd
plugins.

## Tech stack
- java
- maven
  - testng
  - failsafe
  - pmd

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- pmd report found at bin/target/site

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
